## Naming Convention

Experiment directories: `experiments/runs/YYYY-MM-DD_<short-name>/`

## Status Values

- `planned` — config written, not yet run
- `running` — in progress
- `completed` — finished, results recorded
- `failed` — did not complete successfully
- `abandoned` — stopped intentionally, reason documented
